HTML

  • The head element contains info about the webpage.
  • The body element represents the visible content shown to the user.
  • The alt attribute contains a text string that describes the appearance and functionality of the image

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS - style an element directly
  • Internal CSS style sheet - embedded stylesheets - adds rules
  • External CSS style sheet - most commonly used

Git

  • Feature Branch - where new code is added
  • Commit code often.
  • Repo is the abbreviation of repository

JavaScript

  • Control Flow tells the computer the order to execute code.
  • A variable is a named container to store data in.
  • All strings are surrounded by quotation marks.
  • Each line of JavaScript code ends with a semicolon.
